# Codex Dream Skin Studio

Windows-only visual theme editor for the Codex desktop app. It is built with Electron, React, Vite, and TypeScript.

This project is derived from [Fei-Away/Codex-Dream-Skin](https://github.com/Fei-Away/Codex-Dream-Skin). It is not an official OpenAI product and does not modify the Codex application package or `app.asar`.

## Development

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
npm run package:dir
```

Only Windows x64 is supported.
