# 项目栏预览圆角对齐

## 实现范围

Studio 首页预览中的项目选择栏使用 `28px 28px 0 0` 圆角，与实际 Codex 顶部左右圆角一致。底部两个角保持直角，使项目栏继续与下方输入区自然衔接。

本次只修改 Studio 预览样式，不修改实际 Codex 注入逻辑、主题配置、IPC/API 或首页布局尺寸。

## 回归约束

- `tests/preview-layout.test.ts` 必须断言项目栏的四角值，防止顶部圆角退回直角或误将底部圆角化。
- 不得为了视觉对齐读取或复制运行中 Codex 的项目、任务、账号、团队或其他身份数据。
- 源码、测试、文档和验收材料只允许使用公开仓库名或固定虚构数据，并继续遵守 [隐私与示例数据约束](PRIVACY.md)。
- 视觉核对使用用户提供的参考图或离屏测试，不控制用户的前台窗口。

## 验收方式

运行预览布局测试、完整测试、类型检查和生产构建，并确认 `git diff --check` 通过。视觉验收重点检查项目栏顶部左右角为圆角、底部仍为直角，且项目栏宽度和下方输入区位置没有变化。
